# Proceso de Ingreso de Notas

1. **Apertura de Bimestre**: El administrador habilita el periodo.
2. **Ingreso Docente**: Los maestros ingresan notas en `Mi Gestión > Calificaciones`.
3. **Validación**: El sistema valida rangos (0-100).
4. **Cierre**: El administrador cierra el periodo para prevenir cambios.
