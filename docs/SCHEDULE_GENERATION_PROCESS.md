# Proceso de Generación de Horarios

1. **Configuración**: Definir aulas, docentes y cursos.
2. **Preferencias**: Ingresar restricciones de tiempo de docentes.
3. **Generación IA**: Ejecutar el `GeneticScheduler` desde `Admin > Horarios`.
4. **Revisión**: Ajustar conflictos manuales.
5. **Publicación**: Hacer visible para usuarios.
