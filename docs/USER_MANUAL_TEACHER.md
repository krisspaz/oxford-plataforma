# Manual de Usuario: Docente

## 1. Gestión Académica
- **Carga de Notas**: Ingrese a `Docente > Carga de Notas`. Seleccione grado y materia. Ingrese notas numéricas (0-100).
- **Asistencia**: Use `Docente > Asistencia` para marcar presentes/ausentes diarios.

## 2. Comunicación
- **Chat**: Use el módulo de chat para dudas rápidas de estudiantes.
- **Reportes**: Genere reportes de rendimiento de sus clases en `Reportes`.

## 3. Perfil
- Actualice su información de contacto en `Configuración > Mi Perfil`.
