# Manual de Usuario: Estudiante

## 1. Mis Notas
- Vaya a `Estudiante > Mis Notas` para ver su boleta actual.
- Las notas se actualizan en tiempo real cuando el docente las publica.

## 2. Tareas
- Revise `Estudiante > Tareas` para ver asignaciones pendientes.
- Suba archivos PDF o Word como entregables.

## 3. Pagos
- Consulte su estado de cuenta en `Financiero > Pagos`.
- Puede descargar recibos de pagos anteriores.
