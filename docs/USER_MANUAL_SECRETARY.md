# Manual de Usuario: Secretaría

## Introducción
Módulo para la gestión operativa diaria de la institución.

## Funcionalidades
### 1. Inscripciones
- **Nueva Inscripción**: Formulario para registrar nuevos estudiantes.
- **Contratos**: Generación e impresión de contratos de servicios educativos.

### 2. Expedientes
- **Documentación**: Gestión de certificados de nacimiento, antecedentes, y papelería.
- **Constancias**: Generación automática de constancias de estudio.

### 3. Atención
- **Recepción**: Registro de visitas y llamadas.
