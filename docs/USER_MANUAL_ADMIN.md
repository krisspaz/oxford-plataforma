# Manual de Usuario: Administrador

## Introducción
Bienvenido al panel de administración de Corpo Oxford. Este sistema le permite gestionar todos los aspectos de la institución.

## Funcionalidades Principales

### 1. Gestión de Usuarios
- **Crear Usuario**: Vaya a `Admin > Usuarios > Nuevo`.
- **Roles**: Asigne roles (Admin, Docente, Estudiante, etc.) para controlar accesos.
- **Estado**: Active/Desactive usuarios según sea necesario.

### 2. Gestión de Cargos
- **Acceso**: `Admin > Cargos`.
- **Funciones**: Crear nuevos puestos de trabajo, editar jerarquías y asignar departamentos.

### 3. Configuración del Sistema
- **Variables Globales**: Ajuste parámetros como "Ciclo Escolar Activo" o "Límite de Mora".
- **Logs**: Revise el historial de acciones en `Admin > Logs` para auditoría.

## Soporte
Para problemas técnicos, contacte al departamento de TI o use el canal de #soporte.
