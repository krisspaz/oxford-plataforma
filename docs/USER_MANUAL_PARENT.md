# Manual de Usuario: Padres de Familia

## Introducción
Portal de padres para monitoreo académico y financiero.

## Funcionalidades
### 1. Monitoreo Académico
- **Notas**: Vea las calificaciones de su(s) hijo(s) en tiempo real en `Mis Hijos > Notas`.
- **Asistencia**: Verifique reportes de asistencia diaria.
- **Conducta**: Revise observaciones disciplinarias.

### 2. Financiero
- **Estado de Cuenta**: Vea historial de pagos y cuotas pendientes.
- **Pagos**: Instrucciones para realizar transferencias.

### 3. Contacto
- **Directorio**: Contacte directamente con los docentes asignados.
